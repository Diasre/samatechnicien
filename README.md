# Samatechnicien - Application Mobile (Prototype)

Ce projet est une application web React (Vite) simulant l'application mobile Samatechnicien.

## Prérequis

*   **Node.js** (Version 16 ou supérieure) doit être installé sur votre ordinateur.
    *   [Télécharger Node.js ici](https://nodejs.org/)

## Installation

Ouvrez un terminal dans le dossier `samatechnicien` et lancez les commandes suivantes :

1.  **Installer les dépendances** :
    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```

3.  **Lancer le serveur backend** (dans un nouveau terminal) :
    ```bash
    npm run server
    ```

3.  Ouvrez votre navigateur sur l'adresse indiquée (généralement `http://localhost:5173`).

## Fonctionnalités incluses

*   **Accueil** : Hero section, Catégories, Fonctionnalités.
*   **Recherche** : Liste des techniciens filtrable par nom/ville/spécialité.
*   **Profil** : Détails d'un technicien avec boutons de contact simulés.
*   **Vente** : Marketplace de produits d'occasion.
*   **Panier** : Simulation de panier d'achat.
*   **Connexion** : Interface de login (simulation).
*   **Dashboard** : Tableaux de bord Admin et Technicien.

## Technologies

*   React
*   Vite
*   React Router
*   Lucide React (Icônes)
*   CSS Moderne (Variables CSS)
